# LOGIC Health Management Site

A modern healthcare management platform website built with Next.js, React, and Framer Motion.

## Features
- Animated hero section
- Service offerings visualization
- Responsive design
- Contact form

## Tech Stack
- Next.js 15
- React 19
- Framer Motion
- Tailwind CSS

## Running Locally

```bash
npm install
npm run dev
```

Open http://localhost:3000
